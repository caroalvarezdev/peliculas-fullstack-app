# 🎬 Aplicación Full Stack de Gestión de Medios

Este proyecto consiste en una aplicación web full stack para la gestión de contenido audiovisual (películas y series), desarrollada como parte del curso de Ingeniería Web II.

---

## 🚀 Tecnologías utilizadas

### Frontend
- React
- Axios
- Bootstrap

### Backend
- Node.js
- Express
- MongoDB
- Mongoose

---

## 🧠 Descripción del proyecto

La aplicación permite gestionar información relacionada con contenido multimedia, incluyendo:

- Géneros
- Directores
- Productoras
- Tipos (película o serie)
- Medios (películas y series)

Se implementa un CRUD completo con relaciones entre entidades, permitiendo seleccionar información desde listas desplegables y visualizar los datos en una tabla.

---

## 🔗 Características principales

- Creación de registros desde formularios dinámicos  
- Relación entre entidades mediante ObjectId  
- Uso de populate para mostrar datos relacionados  
- Actualización automática de la información en pantalla  
- Arquitectura full stack (frontend + backend)  

---

## ⚙️ Cómo ejecutar el proyecto

### Backend

```bash
cd peliculas-backend
npm install
npm run dev
```

### Frontend

```bash
cd peliculas-frontend
npm install
npm start
```
