# 🎬 Aplicación de Películas - Frontend

Este proyecto corresponde al frontend de una aplicación web desarrollada para la gestión de contenido audiovisual como películas y series.

Permite la interacción con el usuario mediante formularios y tablas para la creación y visualización de información.

---

## ✦ Tecnologías utilizadas

- React.js  
- Axios  
- Bootstrap  

---

## 🌐 Despliegue

La aplicación frontend se encuentra desplegada en:

👉 https://dreamy-shortbread-e13115.netlify.app/

---

## ⚙️ Ejecución local

Para ejecutar el proyecto en entorno local:

npm install
npm start

---

## 🔗 Conexión con el backend

Este frontend consume una API REST desplegada en la nube.

Las rutas utilizadas incluyen:

- /api/generos  
- /api/directores  
- /api/productoras  
- /api/tipos  
- /api/medias  

---

## 🎯 Funcionalidades

- Crear registros desde formularios  
- Visualizar información en tablas  
- Selección de relaciones (género, director, etc.)  
- Visualización de trailers e imágenes
